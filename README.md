숫자야구
1~9로 이루어진 네 자리 숫자를 맞추는 게임
사용자는 네 자리 숫자를 추측하여 입력한다.
컴퓨터는 정답과 입력 받은 숫자를 비교하여 스트라이크와 볼의 수를 알려준다.
입력한 수가 정답과 같은 자리에 있다면 스트라이크, 입력한 수가 정답의 4자리에 존재하고 위치가 다르면 볼이다.
ex) 정답이 1234 일 때
2578을 입력한 경우 : 2가 정답에 존재하고 위치는 다르므로 1볼
3259를 입력한 경우 : 2가 정답과 같은 자리에 있고 3은 정답에 존재하고 위치가 다르므로 1스트라이크 1볼
1259를 입력한 경우: 1,2가 정답과 같은 자리에 있으므로 2스트라이크

9번의 기회가 주어지고 그 안에 정답을 맞추면 게임은 종료. 맞추지 못하면 실패.
